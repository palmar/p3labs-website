Mikrotik Traffic Monitor

Open-source SNMP traffic monitoring

What it does

Mikrotik Traffic Monitor gives you real-time visibility into your Mikrotik router traffic. It's a single Go binary that polls your routers via SNMP and displays live throughput on a web dashboard. No agents, no database, no complex setup.

Features

Single binary, zero dependencies

The entire application — backend, frontend, chart library — compiles into one static binary with embedded assets. Download it, configure your devices, run it.

SNMP v2c and v3

Supports both SNMPv2c and SNMPv3 with full authentication and privacy protocol options. Works with any Mikrotik router without enabling experimental features.

Real-time dashboard

Server-Sent Events stream live throughput data to a dark-themed web UI. Per-interface charts with 20-minute rolling history using fixed-size ring buffers for predictable memory usage.

Auto-discovery

Automatically discovers all interfaces on your routers at startup. Handles unreachable devices gracefully without blocking the rest of your fleet.

Quick start

Create a config.yaml with your device details and run the binary:

devices:
  - name: core-router
    host: 192.168.1.1
    snmp_version: 2c
    community: public

Then open the dashboard in your browser and watch traffic in real time.
