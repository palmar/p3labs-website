NetWatch

Managed network intelligence for SMBs

What it does

NetWatch collects interface statistics from fleets of RouterOS devices and delivers actionable network intelligence to your inbox. No dashboards to check, no agents to install on your routers — just weekly reports with traffic summaries, week-over-week trends, and anomaly alerts.

How it works

RouterOS script-push model

A lightweight RouterOS script runs on your devices and pushes interface statistics to the NetWatch ingest API. No SNMP polling, no port-forwarding — outbound HTTP from your routers is all you need. Supports both HTTPS and HTTP for RouterOS 6.x compatibility.

Anomaly detection

NetWatch tracks interface utilization patterns and flags deviations automatically. Get alerted when a link hits unusual traffic levels before it becomes a problem.

Weekly email reports

Every week, NetWatch sends a structured HTML email with per-interface traffic summaries, week-over-week change analysis, and flagged anomalies. Reports go directly to your inbox — no login required.

REST API

Full API access for device registration, data ingestion, and querying aggregated statistics. Bcrypt-authenticated device keys, rate limiting, and per-minute deduplication built in.

Technical details

  • Go service with PostgreSQL storage
  • Monthly table partitioning for fast queries at scale
  • CLI-driven operations: serve, migrate, aggregate, report, cleanup
  • Rate limiting (2 req/min per device) with deduplication
  • Multi-tenant architecture with customer isolation
  • Email delivery via Resend API

Get started

NetWatch is currently in early deployment. If you manage a fleet of Mikrotik routers and want better visibility into your network traffic, reach out to learn more.
